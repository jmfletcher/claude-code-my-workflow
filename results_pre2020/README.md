# Results: Pre-2020 analysis (excluding COVID-19)

This folder contains a full re-run of the analysis **restricting data to years through 2019**, so that COVID-19 mortality does not affect the estimates.

- **Data:** `Data/processed/` — parquet files with Year ≤ 2019  
- **Figures:** `Figures/` — all pipeline figures from the pre-2020 run  
- **Tables:** `Paper/tables/` — table1–table5 CSVs from the pre-2020 run  
- **Paper:** `Paper/main.md` and `Paper/main.pdf` — draft and PDF for this run  

Generated by running with `OUTPUT_SUBDIR=results_pre2020` and `END_YEAR=2019`.
