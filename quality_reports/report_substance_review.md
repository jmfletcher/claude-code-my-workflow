#### Issue 1.2: Table 3 Milwaukee Black 95% CI lower bound

- **Location:** Table 3 (Annual deaths, minimum detectable effect)
- **Severity:** MINOR
- **Claim:** Milwaukee Black: "95% CI 41–70"
- **Problem:** The script computes the count-based CI as D ± 1.96√D. For D=55: 55 − 14.54 ≈ 40.46. The script outputs 40–70 (rounding 40.46 down to 40). The report shows 41–70.
- **Suggested fix:** Use 40–70 to match the script output, or document that the report uses a different rounding convention.

### Verified as correct

- **Rate formula:** rate = 1000 × D/B ✓
- **CI formula:** rate ± 1.96 × (1000√D)/B ✓
- **CI lower bound clamping:** Script uses `max(0, rate - half)` ✓
- **Rest of Wisconsin:** State − Milwaukee − Dane ✓ (verified against `data/processed/` CSVs)
- **Table 1 period:** 2019–2023 ✓
- **Detectability formula:** min averted = ceil(1.96√D) ✓
- **NFP spend:** min averted × $3.2M ✓ (e.g., Milwaukee Black: 15 × 3.2 = 48)
