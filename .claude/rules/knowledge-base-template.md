## Cohorts / "treated" weeks (confirmed from legacy data + cohort profiles)

| Cohort | Full study name | Treated birth week | `group` code | Legacy `cohort=1` |
|--------|-----------------|--------------------|-------------|-------------------|
| NSHD 1946 | MRC National Survey of Health & Development | **3–9 March 1946** | 1 | row 23 in `final.csv` |
| NCDS 1958 | National Child Development Study | **3–9 March 1958** | 2 | row 68 in `final.csv` |
| BCS70 1970 | 1970 British Cohort Study | **5–11 April 1970** | 3 | row 113 in `final.csv` |

**Control definition:** 8 adjacent birth weeks per cohort group, same calendar cluster.
(`cohort = 0` for all control rows in legacy CSVs; 9 weeks × 3 groups = 27 birth-week series.)

**Geography:** England and Wales (ONS administrative data).
The 1946 NSHD originally included Scotland — note this boundary in methods.

**Data coverage:** ONS deaths 1970–2013; birth years in sample approximately 1944–1972.

## Design principles

| Principle | How we enforce it |
|-----------|-------------------|
| Denominators explicit | Every rate traces to documented N; sanity checks on totals |
| Treated/control stable | `cohort` and `group` codes fixed; no ad hoc relabeling |
| Publication figures | Single seaborn/matplotlib theme; vector PDF for print; PNG for slides |
| Reproducibility | `python3 analysis/00_data_overview.py` → `01` → `02` → `03` → `04` documented in README |

---

## Anti-patterns

| Anti-pattern | Risk | Correction |
|--------------|------|------------|
| Plotting rates without showing N | Misleading precision | Table/caption with denominators |
| Mixing England & Wales with UK | Wrong comparability | Flag in methods; stratify if needed |
| Editing CSVs by hand | Broken lineage | All transformations in `analysis/01_load_and_clean.py` |
| Hard-coded absolute paths in Python | Breaks on other machines | Use `pathlib.Path(__file__).resolve().parent.parent` |

## Common pitfalls (extend as discovered)

| Issue | Fix |
|-------|-----|
| `xlrd` can only read `.xls` (not `.xlsx`) | Use `engine="xlrd"` for ONS `.xls` files |
| Week-of-year edge cases (week 53) | Confirm against ONS documentation |
| `week_in_year` is cluster-relative (1–9), not ISO week | Do not compare across groups without remapping |
