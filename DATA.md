# Data Access Instructions

Raw data files are **not included** in this repository. This document describes how to
obtain them and where to place them.

---

## 1. ONS deaths by week of birth (England and Wales, 1970–2013)

**Source:** Office for National Statistics (ONS), *Deaths registered by cause of death
and week of birth, England and Wales, 1970–2013*.

**Files needed:**

| Filename | Description |
|----------|-------------|
| `deathsregisteredbymonthbyselectedweekofbirthenglandandwales1970to2013_tcm77-419401.xls` | Total death counts by year, week of birth |
| `deathsregisteredbycauseofdeathicdchapterbyselectedweekofbirthenglandandwales1970to2013_tcm77-419405.xls` | Cause-specific death counts (ICD chapters) |

**Access:** These files were obtained directly from ONS. The dataset is part of ONS's
custom tabulations for researchers. To request access, contact ONS via:
<https://www.ons.gov.uk/aboutus/whatwedo/statistics/requestingstatistics>

**Placement:** Place both `.xls` files in `Data/`:

```
Data/
├── deathsregisteredby...tcm77-419401.xls
└── deathsregisteredby...tcm77-419405.xls
```

---

## 2. Human Mortality Database — England and Wales (GBRTENW)

**Source:** Human Mortality Database (HMD), University of California, Berkeley (USA) and
Max Planck Institute for Demographic Research (Germany). Available at
<https://www.mortality.org>.

**File needed:** `Mx_1x1.txt` — period death rates (mx) by single year of age and
calendar year for England and Wales (population code `GBRTENW`).

**Access:** Free registration required at <https://www.mortality.org/Account/Auth>.
After logging in, navigate to *England and Wales Total > Death rates > Period > 1x1*
and download `Mx_1x1.txt`.

**Placement:** Place the file at:

```
Data/HMD/Mx_1x1.txt
```

---

## 3. Denominator construction

Raw ONS files contain death counts, not birth counts. The analysis constructs mortality
rates using an imputed birth-week cohort size of $N_b \approx 15{,}334$ per birth week.
This is derived from historical ONS live-birth registration statistics for England and
Wales (approximately 800,000–850,000 births per year in the 1940s–1970s, implying
roughly 15,000–16,000 per week). The denominator is constant across all weeks in the
regression; since birth-week fixed effects are included, inference on the treatment
coefficient is invariant to the common scale factor provided cohort size does not differ
systematically between treated and control weeks.

---

## 4. Pre-computed outputs

All intermediate tables and figures generated by the analysis pipeline are tracked in
this repository under `output/figures/` and `output/tables/`. You can inspect all
results and read the manuscript (`manuscript/main.pdf`) without downloading the raw data.
The raw data are only needed if you wish to re-run the pipeline from scratch.

---

## 5. Legacy files (reference only)

The `Old Attempts and Results/` folder contains earlier partial analyses in Stata.
These are **not** part of the current analysis pipeline and are included for
reference only. Script `02_replicate_stata.py` ports the legacy regression
specifications to Python to verify consistency.
