# Manual GitHub upload helpers

Use **`pdf_bundle/`** when you want a small, ready-to-upload folder with the latest rendered manuscript (`paper.pdf` and optional `paper.html`).

1. From the repo root, refresh the bundle: `bash scripts/package_pdf_bundle_for_github.sh`
2. Upload everything inside `docs/manual_github_upload/pdf_bundle/` via the GitHub website (Add file → Upload files).

See `pdf_bundle/README.md` for details.
